Storyo

用您自己的声音录制睡前故事，添加AI插图，为孩子们带来神奇的故事书体验——无论您身在何处。

📱 Android & Web · v1.0
🌟
概述
应用程序的功能及使用方式

Storyo 是一款面向家庭的个人有声读物应用。 父母、祖父母或看护人用自己的声音录制一个故事，可选择添加AI生成的插图，孩子便能像看精美动画绘本一样播放它——即使讲故事的人不在身边。

🎙️
录制你的声音

以30秒为单位分段录制故事。如果某段出错，可单独重录，其他段落保持不变。

🎨
AI插图

场景随故事播放自动出现。可使用Claude SVG、DALL-E 3或GPT Image 1，也可上传自己的图片。

📖
电影级播放器

插图支持肯·伯恩斯平移缩放效果，场景间平滑淡入淡出，配有进度条和场景指示点。

📦
分享故事

导出单个 .storybook 文件——音频、图片和元数据一并打包——通过WhatsApp、邮件或任何文件共享应用发送。

💡
无需账号。

所有内容均存储在设备本地。无需登录、订阅或云同步。

🏠
主屏幕
主故事库

主屏幕以网格磁贴形式显示所有已保存的故事。每块磁贴显示故事的表情符号、标题以及讲述者的彩色标记。

讲述者颜色编码

每块磁贴顶部的彩色条纹和底部的标记表示谁录制了这个故事：

👨 爸爸 — 蓝色
👩 妈妈 — 粉色
👴 爷爷 — 绿色
👵 奶奶 — 红色

打开故事

点击任意故事磁贴即可立即打开故事播放器。音频加载后，插图会随播放进度自动出现。

底部栏

底部栏有两个按钮：

  • 壁纸按钮（左）——打开背景主题选择器
  • 设置 (⚙️)（右）——打开PIN保护的管理区域
🎨
壁纸
六种动态背景主题

点击主屏幕左下角的壁纸按钮可选择背景主题。选择会自动保存并在会话间保持。

🌸 女孩
🚀 男孩
🦄 独角兽
🌲 森林
🌊 海洋
🌙 夜晚

每个主题都有专属动态背景——飘落的花瓣、闪烁的星星、漂移的云朵、摇曳的树木、海浪等。标题和文字颜色会自动适配，确保在浅色和深色主题下均清晰可读。

⚙️
设置 & PIN码
PIN码保护的管理区域

点击⚙️ 设置会打开PIN输入界面，防止孩子意外编辑或删除故事。

默认PIN码

⚠️
默认PIN码为1234。

首次使用后请更改它——点击设置屏幕右上角的更改PIN。

更改PIN码

  1. 1
    输入当前PIN码输入现有的4位PIN码进行验证。
  2. 2
    输入新PIN码选择任意4位数字。
  3. 3
    确认新PIN码再次输入。两次输入一致后，PIN码立即更新。

设置界面内容

进入后，设置界面显示：

  • AI插图设置卡片
  • ✨ 新故事——创建新故事
  • 📥 导入故事——从其他设备导入 .storybook 文件
  • 所有已保存故事的完整列表，每个故事均有编辑、导出和删除按钮
✨
新故事
创建和编辑故事

在设置中点击✨ 新故事打开故事编辑器。编辑器也用于修改已有故事——点击列表中任意故事旁的编辑即可。

讲述者

选择讲故事的人。这将决定主屏幕磁贴的颜色编码。

标题

输入故事名称。至少输入一个字符后，保存按钮才会激活。

表情符号

从包含80多种动物和奇幻角色的网格中选择一个表情符号。它将作为主屏幕磁贴的大图标，以及插图加载前播放器中的占位图。

保存

点击右上角的已保存 ✓。写入存储时按钮显示保存中...，完成后恢复为已保存 ✓。每次点击按钮都会自动保存更改——编辑过程中随时可点击。

💡
无自动保存。

只有当您点击已保存 ✓时，音频片段和场景才会写入存储。为了安全起见，每次录制片段后都要点击它。

🎙️
录音
分段录音的工作原理

音频以30秒为单位分段录制。故事由多个片段组成，播放时无缝衔接。

  1. 1
    点击 🎙️ 录制片段录制立即开始。红点闪烁，进度条显示30秒已用时间。
  2. 2
    朗读故事最后5秒面板快速闪烁提示。录制在30秒时自动停止。
  3. 3
    点击完成 ✓ 提前结束无需用满30秒。随时点击完成 ✓ 结束本段录制。
  4. 4
    回听并继续完成的片段以卡片形式显示时长。点击 ▶ 预听。
  5. 5
    重复录制后续片段依次点击录制片段2、3等。片段数量不限。

重录片段

每个片段卡片上都有🔄 重录按钮。点击后仅替换该片段，其他片段保持不变。

删除片段

点击任意片段卡片上的✕按钮即可删除。其余片段顺序不变。

预听片段

点击任意片段卡片上的▶，可在编辑器中直接预听，无需打开完整播放器。

⚠️
录音后请保存。

添加片段后点击已保存 ✓。若不保存就离开编辑器，新录音将会丢失。

🖼️
插图
为故事添加图片

插图会在您设定的时间戳自动出现。每张插图绑定到一个时间点——播放到达该点时，图像以流畅的肯·伯恩斯平移缩放效果淡入。

添加场景

  1. 1
    点击 + 添加出现新的场景行，包含时间戳字段和提示词字段。
  2. 2
    设置时间戳输入该插图应出现的时间（格式为 分:秒，如 0:30）。也可仅输入秒数，如 45。
  3. 3
    编写AI提示词或上传图片向AI描述场景，或点击提示词字段下方的图片按钮上传自己的图片。

AI生成的插图

配置AI提供商后，首次播放故事时会自动生成插图。生成过程中应用显示🎨 正在绘制场景...。

ℹ️
插图会被缓存。

生成后，插图存储在设备上，后续每次播放均直接复用。AI提供商仅对首次生成收费。

自定义（上传）图片

点击提示词字段下方的图片区域，可从设备上传照片或图片。上传的图片优先于AI生成——有图片时AI提示词将被忽略。

点击图片缩略图上的删除，可恢复使用AI生成该场景的插图。

场景指示点

播放器中，进度条下方出现小圆点——每个场景对应一个。当前激活的圆点会展开，显示正在播放的插图。

🤖
AI设置
选择提供商并输入API密钥

AI插图为可选功能，需要您向支持的提供商申请自己的API密钥。密钥仅存储在您的设备上，除直接发送给AI提供商的API外，不会传输至任何其他地方。

支持的提供商

提供商 模型 输出格式 估算费用/图 质量

🖼️ DALL-E 3

获取API密钥

Anthropic (Claude SVG)

访问 console.anthropic.com → API Keys → Create Key

密钥以 sk-ant- 开头

OpenAI (DALL-E 3 / GPT Image 1)

访问 platform.openai.com → API Keys → Create New

密钥以 sk- 开头

在应用中配置

  1. 1
    打开设置 (⚙️)输入PIN码进入管理区域。
  2. 2
    点击AI插图卡片卡片展开，显示提供商选择器和密钥字段。
  3. 3
    选择提供商选择Claude SVG、DALL-E 3或GPT Image 1。
  4. 4
    粘贴API密钥密钥已隐藏（密码字段）。点击保存。
💡
没有AI密钥？

您仍然可以完整使用应用程序——只需上传自己的图片作为插图，而不是使用AI生成。

▶️
故事播放器
全屏播放体验

点击主屏幕上的任意故事磁贴即可打开播放器。播放器加载音频后，随故事进度自动显示插图。

控制按钮

▶ / ⏸ 播放 & 暂停

中央大按钮。点击开始或暂停播放。

⏪ / ⏩ 跳转

向前或向后跳转10秒，便于重听某个片段。

进度条

点击或拖动进度条上的任意位置跳转至故事对应时刻。场景切换标记以进度条上方的小竖线显示。

✕ 关闭

点击 ✕ 按钮或点击播放器卡片外部，关闭并返回主屏幕。

插图行为

播放到场景时间戳时，新插图在1.4秒内淡入，同时前一张淡出。插图随后缓慢平移缩放（肯·伯恩斯效果），直到下一次场景切换，营造电影质感。

若插图尚未生成，应用显示🎨 正在绘制场景...并在后台生成。预加载下一场景插图时，会显示小标记🎨 正在绘制下一张…。

ℹ️
生成仅发生一次。

插图在首次播放时生成并永久缓存。再次播放故事即时呈现，无需重复调用API。

生成失败时

若AI提供商返回错误，插图区域会显示警告和🔄 重试按钮。点击重试将从当前播放位置重新生成插图。

📦
导出 & 导入
在设备间传输故事

故事可导出为单个 .storybook 文件，并导入到任何安装了该应用的设备。文件包含所有内容：音频、插图、场景时间戳和元数据。

导出故事

  1. 1
    打开设置 (⚙️)输入PIN码。
  2. 2
    找到故事在列表中滚动到该故事。
  3. 3
    点击导出在Android上，文件直接保存到下载文件夹。在网页端，下载自动开始。

导入故事

  1. 1
    接收 .storybook 文件通过WhatsApp、Telegram、邮件或任何文件传输方式接收，并保存到设备。
  2. 2
    打开设置 (⚙️)输入PIN码。
  3. 3
    点击 📥 导入故事文件选择器打开。选择 .storybook 文件。
  4. 4
    故事出现在列表中若已存在同名故事，系统会询问是否作为副本导入。

.storybook 文件格式

_type: "storybook/v1"
exportedAt: "2025-01-15T20:30:00Z"
meta: { title, narrator, emoji, colorIdx, scenes }
segments: [ base64 WAV audio per segment ]
svgs: { 已缓存的AI插图 }
sceneImgs: [ uploaded custom images ]
⚠️
文件大小。

含多张AI插图的5分钟故事可达10–25 MB。建议使用Telegram或Google Drive传输——WhatsApp会压缩文件，可能导致损坏。

分享故事
将故事传输到孩子的设备

适合家庭双设备使用的推荐流程——一台用于录制，一台供孩子收听：

  1. 1
    在家长设备上录制用家长的手机创建并保存故事。
  2. 2
    导出 .storybook 文件在设置中点击导出。文件保存到下载文件夹（Android）或在浏览器中下载。
  3. 3
    发送文件
  4. 4
    在孩子的设备上导入打开应用，进入设置，点击 📥 导入故事，选择收到的文件。
  5. 5
    播放故事出现在主屏幕。点击即可收听。

存储 & 隐私
数据存储位置

所有数据通过 IndexedDB 存储在设备本地。无需账号，无云同步，无服务器。

存储内容

stories-meta — 标题、讲述者、表情符号、场景列表
segments:{id} — 每个故事的base64 WAV音频
scene-imgs:{id} — 每个故事上传的自定义图片
svg-cache — 已生成的AI插图（所有故事）
api-key — 您的AI提供商API密钥
ai-provider — 已选择的AI提供商
wallpaper — 已选择的背景主题
app-pin — 4位管理员PIN码
🔒
API密钥隐私。

您的API密钥直接从您的设备发送给AI提供商（Anthropic或OpenAI）。它永远不会发送到任何其他服务器。该应用没有后端。

存储限制

IndexedDB没有像localStorage那样的5 MB硬性限制——可使用设备可用存储空间的相当大一部分。含音频和多张AI插图的典型5分钟故事约占15–30 MB。

💡
技巧 & 窍门
充分发挥应用的潜力

录音质量

  • 在安静的房间录音——背景噪音会被清晰录入
  • 以正常说话距离持握手机，不要太近
  • 30秒分段提供了自然停顿点——围绕这些节点规划故事
  • 对单个片段使用重录按钮，而非重新录制整个故事

AI提示词编写

  • 描述场景中正在发生的事，而非仅列出存在的物体："一只小兔子穿越月光下的草地"比"兔子，草地"效果更好
  • 描述氛围和光线："温暖的金色午后阳光"、"阴森的黑暗森林"
  • 提示词控制在1–2句——更长的提示词不一定带来更好的效果
  • GPT Image 1生成最细腻的绘画风格图像；Claude SVG适合更简洁的风格化场景

场景时间设置

  • 场景时间戳设置在希望图片出现时刻的前一两秒——淡入效果约需1.4秒
  • 设置在 0:00 的场景会从播放一开始就显示插图
  • 典型的2–3分钟故事使用3–5个场景——过多会显得拥挤

发送文件

  • Telegram — 最佳选择，可无压缩发送最大2 GB的文件
  • Google Drive / Dropbox — 大文件传输稳定可靠
  • 电子邮件 — 适合25 MB以内的文件
  • WhatsApp — 会压缩媒体文件，可能导致导入错误。仅在其他方式不可用时使用
